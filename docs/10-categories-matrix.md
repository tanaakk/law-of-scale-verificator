# 10カテゴリー「地球文化・3大勢力」整合性マトリクス

## 概要

デザインを「出荷」する前に、以下の10領域において、人類の基底OS（三大勢力）との衝突がないか、あるいはそれらを包含しているかをチェックする。

|----------|------|------------|-------------|
